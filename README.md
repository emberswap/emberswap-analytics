Analytics site for [EmberSwap DEX](https://emberswap.com).

### To Start Development
Have nvm installed 
```bash
nvm use
npm install yarn -g
```
###### Installing dependencies
```bash
yarn
```

###### Running locally
```bash
yarn start
```
